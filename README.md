# Comfort Transit Bot 🚌

An AI-powered chatbot assistant that helps daily commuters with public transit queries — route planning, schedules, and smart travel tips.

## 🚀 Features

- Conversational chatbot interface for transit queries
- Route planning and schedule assistance
- Real-time commute guidance
- Smart travel comfort tips
- Clean and responsive React UI

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React.js, TypeScript, Tailwind CSS |
| AI Layer | Conversational AI / NLP |
| Build Tool | Vite |

## ⚙️ How to Run

```bash
git clone https://github.com/dharineesh-812/comfort-transit-bot.git
cd comfort-transit-bot
npm install
npm run dev
```

## 💬 Sample Queries

- "What is the best route from Singanallur to RS Puram?"
- "Is the Coimbatore city bus running on Sundays?"
- "How do I get to Ukkadam bus stand from Gandhipuram?"
